Projects

What I've been up to

This page is updated from time to time with some things I've built. Check out my Github for more of what I'm currently working on!

Yummy: Online Order and Reservation System : This project is a full-stack project based on Python, JavaScript and MySQL. It's an online restaurant reservation and order system deployed with AWS EC2. It is available for use by real business with options like adding and deleting dishes, registering staffs, and managing reservations. Link to this website.

Spatial-Temporal Extraction and Analytics Framework : This project is a web-based data analytics framework based on Java and TypeScript to extract and display time and space data using NLP techniques from various sources including tabular data and plain texts. We designed it using strategy pattern to keep its extensibility for easily adding new analytic plugins.

Product Review Keywords Extraction : This project is an NLP project based on Python to extract keywords from product reviews. These keywords come in triplets of 'aspect', 'opinion', and 'sentiment' to help users understand what people are talking about the product and how they feel about it. We made some improvements on the original algorithm to make it more accurate and efficient. Here for more information.
